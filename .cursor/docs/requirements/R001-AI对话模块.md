# R001 - AI 对话模块

## 需求概述

基于阿里云百炼大模型，实现多轮 AI 机器人对话能力，支持流式输出与会话管理。

## 功能要点

- 聊天界面（用户消息 / AI 回复）
- 流式打字效果
- 新建 / 切换 / 历史会话
- BFF 代理调用百炼 API（不暴露 API Key）

## 验收标准

- [ ] 输入消息后可获得 AI 回复
- [ ] 回复以流式逐字展示
- [ ] 支持新建会话、切换历史会话
- [ ] 多轮对话保持上下文
- [ ] 错误时有友好提示

## 技术备注

- 百炼 OpenAI 兼容接口：`https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions`
- 流式：`stream: true`
- 会话状态用 Pinia 管理
