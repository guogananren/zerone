# 技术选型说明

## 前端

| 技术 | 选型 | 理由 |
|------|------|------|
| 框架 | Vue 3 | 组合式 API 与 Zerone 模块化理念契合 |
| 构建 | Vite | 快速 HMR，开发体验好 |
| 语言 | TypeScript | 类型安全，便于 API 与状态管理 |
| 状态 | Pinia | 轻量、与 Vue 3 官方推荐 |

## 架构

| 决策 | 说明 |
|------|------|
| BFF 中间层 | API Key 不暴露在前端，由 Node 服务代理百炼请求 |
| 模块解耦 | 对话、代码生成、知识库按功能拆分，便于扩展 |

## AI 接入

- **平台**：阿里云百炼（DashScope）
- **接口**：OpenAI 兼容模式，支持流式输出
- **模型**：可配置，默认通义千问系列
