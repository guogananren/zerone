# Zerone 项目搭建文档

## 一、技术栈与版本

| 技术 | 版本 |
|------|------|
| Vue | ^3.5 |
| Vite | ^7.3 |
| Vue Router | ^5.0 |
| Pinia | ^3.0 |
| Element Plus | ^2.13 |
| TypeScript | ~5.9 |
| Scss | ^1.97 |
| Axios | ^1.13 |

## 二、依赖列表

### 生产依赖

- vue、vue-router、pinia
- element-plus
- axios

### 开发依赖

- vite、@vitejs/plugin-vue
- typescript、vue-tsc
- sass
- eslint、eslint-plugin-vue、eslint-plugin-oxlint
- vite-plugin-vue-devtools

### 包管理

- pnpm

## 三、目录结构

```
zerone/
├─ public
├─ src
│  ├─ common          # 通用目录
│  │  ├─ apis
│  │  ├─ assets
│  │  ├─ components
│  │  ├─ composables
│  │  ├─ constants
│  │  └─ utils
│  ├─ http            # 网络请求
│  ├─ layouts         # 布局
│  ├─ pages           # 页面
│  │  └─ home
│  ├─ plugins         # 插件
│  ├─ router
│  ├─ stores
│  ├─ App.vue
│  └─ main.ts
├─ .env
├─ .env.development
├─ .env.production
├─ vite.config.ts
├─ tsconfig.json
└─ package.json
```

### 暂未创建的目录（按需添加）

- `.husky` - Git hooks
- `tests` - 单元测试
- `src/pages/login` - 登录模块
- `uno.config.ts` - UnoCSS
- `.env.staging` - 预发布环境

## 四、路径别名

- `@` 指向 `src`
- `@@` 指向 `src/common`

## 五、常用命令

| 命令 | 说明 |
|------|------|
| `pnpm dev` | 启动开发服务器 |
| `pnpm build` | 构建生产版本 |
| `pnpm preview` | 预览构建结果 |
| `pnpm lint` | 代码检查与格式化 |

## 六、创建前准备

使用 create-vue 或开发前，建议执行：

```bash
nvm use 20.19
```

以确保 Node 版本满足要求（^20.19.0 或 >=22.12.0）。
