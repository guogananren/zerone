# 01 - 阶段一原型

## 功能清单

| 序号 | 功能 | 说明 | 状态 |
|------|------|------|------|
| 1 | 工作台首页 | 入口页，展示模块入口 | 待实现 |
| 2 | AI 对话 | 聊天界面、流式输出、会话管理 | 待实现 |
| 3 | 自然语言生成代码 | 输入需求描述，AI 返回代码 | 待实现 |
| 4 | 代码预览 | 展示代码、复制、简单编辑 | 待实现 |

## 页面结构

- `/` - 工作台首页
- `/chat` - AI 对话
- `/codegen` - 代码生成

## 关键交互

1. **AI 对话**：输入 → 流式展示回复 → 新建/切换会话
2. **代码生成**：输入需求 → 流式生成代码 → 预览/复制

---

## 交互流程与关键状态

### 工作台首页 (`/`)

| 状态 | 说明 |
|------|------|
| 初始 | 居中展示 Zerone 标题、副标题、2 个模块卡片 |
| 悬停 | 卡片边框/背景轻微变化，传递可点击 |
| 点击 | 跳转至对应模块 |

### AI 对话页 (`/chat`)

| 状态 | 说明 |
|------|------|
| 空会话 | 引导文案「输入消息开始对话。支持流式输出与多轮上下文。」 |
| 输入中 | 输入框聚焦，Enter 可发送 |
| 生成中 | 展示流式打字效果，发送按钮禁用并显示「生成中...」 |
| 多会话 | 左侧列表展示历史会话，点击切换；「新对话」创建新会话 |
| 错误 | 消息区展示错误文案，可重试 |

### 代码生成页 (`/codegen`)

| 状态 | 说明 |
|------|------|
| 空态 | 需求输入框占位「描述你想实现的功能」，代码区为空或占位 |
| 生成中 | 代码区流式展示生成内容 |
| 完成 | 展示完整代码，可复制；可继续输入新需求 |
| 错误 | 代码区或上方展示错误提示，可重试 |
