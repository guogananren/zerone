# 00 - 项目概述

## 项目定位

**Zerone** —— 从 0 到 1 的 AI 辅助开发平台。围绕「用自然语言描述愿景，AI 生成/执行，快速迭代」理念构建，助力开发者将想法转化为可运行代码。

## 核心价值

**Zerone 核心理念**：用自然语言描述愿景 → AI 生成/执行 → 快速迭代，开发者接受代码而不必完全理解细节，实现从 0 到 1 的创造闭环。

**面试亮点**：
- 体现对 AI 辅助开发趋势的理解
- 全栈能力、模块化架构设计
- 「用 AI 做 AI 产品」的闭环

## 技术栈

- **前端**：Vue 3 + Vite + TypeScript
- **状态管理**：Pinia
- **AI 能力**：阿里云百炼大模型（DashScope API）
- **架构**：BFF 中间层（API Key 安全代理）
